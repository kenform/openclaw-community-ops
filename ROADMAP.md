# ROADMAP (30 days)

## Week 1 — Stability baseline
- [ ] Keep no-GPT timers stable for 7 days
- [ ] Zero critical errors in group cleanup reports
- [ ] Finalize Discord server skeleton (channels/roles/pins)

## Week 2 — Community operations
- [ ] Launch subscriber chat workflow (rules + navigator + moderation cadence)
- [ ] Weekly stream format v1 (agenda + follow-up notes)
- [ ] Monthly stream plan (deep-dive)

## Week 3 — Content system
- [ ] 7-day content cadence pilot
- [ ] Start 30-day rolling content board
- [ ] Weekly feedback loop from subscribers (manual, forms paused)

## Week 4 — Release-ready ops
- [ ] Issue labels/process hygiene in GitHub
- [ ] Incident playbook for parser/service failures
- [ ] Monthly review report: what worked / what to improve

## Definition of success
- Stable autonomous ops without GPT dependency
- Clear community cadence (content + support + streams)
- Predictable release process in GitHub
