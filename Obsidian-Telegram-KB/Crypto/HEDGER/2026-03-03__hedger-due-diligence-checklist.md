# HEDGER — due diligence чеклист перед масштабированием

## Данные
- [ ] Есть история сделок минимум за 30 дней
- [ ] Доходность посчитана в net-формате (после комиссий)
- [ ] Учтены довнесения/выводы в equity-кривой

## Риск
- [ ] Зафиксирован max drawdown
- [ ] Определён лимит риска на день/неделю
- [ ] Есть stop-условия отключения стратегии

## Исполнение
- [ ] Понятна ликвидность на рабочих парах/инструментах
- [ ] Измерено среднее проскальзывание
- [ ] Проверен результат на увеличенном объёме (stress test)

## Операционка
- [ ] Есть лог ошибок/сбоев
- [ ] Есть план действий при отказе/деградации
- [ ] Есть еженедельный отчёт в едином формате
