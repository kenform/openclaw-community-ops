# Obsidian Telegram KB

Структура для разбора Telegram-каналов/групп и подготовки чистого контента.

## Папки
- `00_Inbox` — сырые выгрузки и первичные заметки
- `10_Channels` — по папке на канал/группу
- `20_Summaries` — дайджесты и резюме
- `30_FAQ` — ответы на частые вопросы
- `40_Published` — опубликованные версии постов
- `90_MOC` — карты навигации
- `99_Templates` — шаблоны заметок

## Рекомендуемый формат внутри `10_Channels`
`10_Channels/<название_канала>/`
- `raw/` — импорт/сырые заметки
- `clean/` — очищенные заметки
- `timeline.md` — хронология
- `index.md` — главная заметка канала
