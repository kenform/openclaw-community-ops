# Launch Kit: Урок №1 (публикация)

## Позиционирование
- Тема: «Как получать рабочий код от ИИ с первого-второго запроса»
- Формат: короткий практический урок (10–15 минут)
- Цель: быстрый результат для новичка + прогрев к платному курсу

## 1) Скрипт урока №1
### Хук (0:00–0:30)
«90% проблем не в ИИ, а в кривом запросе. Сейчас покажу формулу, которая экономит часы.»

### Суть (0:30–4:00)
Формула 5 блоков:
1. Цель
2. Стек
3. Ограничения
4. Формат ответа
5. Критерий проверки

### Демо (4:00–10:00)
Промпт:

> Я новичок. Нужна страница с заголовком, 3 карточками и кнопкой.
> Стек: html/css/js.
> Ограничения: без библиотек, простой код.
> Выведи по файлам + чеклист проверки.

Проверка:
- есть `index.html`, `style.css`, `script.js`
- страница открывается
- элементы отображаются
- кнопка работает

### Финал (10:00–12:00)
«Если ответ плохой — не начинай заново, уточняй блоками. Это и есть вайбкодинг без хаоса.»

## 2) Шаблон поста-анонса
### Заголовок
Урок 1: Как писать промпты, чтобы ИИ выдавал рабочий код

### Тело
- Формула 5 блоков (без воды)
- Реальный мини-кейс за 15 минут
- Чеклист “как понять, что сделано правильно”

### CTA
«Хочешь шаблон промпта + мини-домашку? Пиши `УРОК1` в комменты.»

## 3) Домашка + чеклист
### Домашка
Сделай простую страницу (заголовок + 3 карточки + кнопка) строго по формуле 5 блоков.

### Что сдать
- твой промпт
- скрин результата
- где застрял (если было)

### Чеклист
- [ ] есть 3 файла
- [ ] всё открывается без ошибок
- [ ] визуал читаемый
- [ ] можешь объяснить, что сделал ИИ и что проверил сам
