# Паттерн поста: «Итоги недели + масштабирование депозита»

## Контекст
Пост повторяет формат предыдущих отчётов:
- публичные цифры по неделе
- тезис о росте депозита
- ожидание следующего отчёта
- CTA на комментарии/реф-ссылки

## Паттерн, который работает
1. Серийность (неделя 1/2/3) = удержание внимания.
2. Цифры = ощущение прозрачности.
3. Тизер следующей недели = сильный retention-hook.
4. Комменты с вопросами = вовлечение и будущий контент.

## Где уязвимость доверия
1. Любая арифметическая неточность мгновенно бьёт по репутации.
2. Нет риск-блока (просадка, комиссии, liquidity risk).
3. Реферальная часть выглядит агрессивно без строгих дисклеймеров.

## Как улучшить этот формат (для твоего канала)
- Добавить мини-таблицу в каждый отчёт:
  - Start balance
  - End balance
  - Net PnL
  - Fees
  - Max drawdown
- Добавить отдельный блок: «что могло пойти не так».
- Разделить educational value и affiliate CTA визуально и логически.

## Вывод
Формат рабочий для роста и удержания, но долгосрочно побеждает только при максимально чистой методике и прозрачной математике.
